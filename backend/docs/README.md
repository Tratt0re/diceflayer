# Diceflayer (BE) - Documentation

![Diceflayer Logo](../../assets/diceflayer_logo.png)

[Diceflayer](https://t.me/MrDiceflayerBot) is a Telegram Mini App designed to assist users in rolling dices for tabletop games like Dungeons & Dragons. **This repository contains the backend code for the Diceflayer telegram mini-app.**
